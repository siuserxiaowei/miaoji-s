Codex 小白上手包

小白用 Codex 时怎么用得更好？先装这 10 个 Skills

Skills 不是多一个按钮，而是给 Codex 的工作说明书。装好以后，小白只要把任务说清楚， Codex 就能按对应流程做设计、开发、测试、审查、修复、文档和 PR。

01

frontend-design

来源：anthropics/skills

已安装到 ~/.agents/skills

安装命令

npx -y skills add anthropics/skills -g --skill frontend-design -y --full-depth

小白一句话

“帮我把这个页面做得像真实产品，不要通用 AI 风格。”

什么时候用

做页面、组件、落地页、后台界面、HTML 原型，或者觉得当前 UI 太模板化时。

让 Codex 多问一句

让它先确认用途、受众、气质和技术约束，再动手写界面。

最适合小白的结果

你不用懂设计语言，也能拿到更完整的布局、字体、颜色、状态和交互。

安全扫描：Safe / Low Risk
02

cache-components

来源：vercel/next.js

已安装到 ~/.agents/skills

安装命令

npx -y skills add vercel/next.js -g --skill cache-components -y --full-depth

小白一句话

“这是 Next.js 项目，帮我检查缓存边界和 Server Component 写法。”

什么时候用

Next.js 项目里用了 cacheComponents、PPR、Server Components，或者页面性能和缓存行为不稳定时。

让 Codex 先检查

先看 next.config 里有没有 cacheComponents，再决定是否使用 use cache、cacheLife、cacheTag。

最适合小白的结果

不用一上来背缓存概念，让 Codex 帮你判断哪些内容该静态、哪些该动态。

安全扫描：Safe / Low Risk
03

fullstack-developer

来源：Shubhamsaboo/awesome-llm-apps

已安装到 ~/.agents/skills

安装命令

npx -y skills add Shubhamsaboo/awesome-llm-apps -g --skill fullstack-developer -y --full-depth

小白一句话

“帮我从前端、API、数据库一起拆方案，并做一个最小可用版本。”

什么时候用

你想做一个完整功能，但不知道要不要建表、接口怎么设计、前端状态怎么接。

让 Codex 先拆层

让它分成数据模型、接口、页面、状态、验证、部署几个部分，再开始实现。

最适合小白的结果

避免只做出一个壳子，能让 Codex 一次性考虑前后端闭环。

安全扫描：Low Risk / Low Risk
04

frontend-code-review

来源：langgenius/dify

已安装到 ~/.agents/skills

安装命令

npx -y skills add langgenius/dify -g --skill frontend-code-review -y --full-depth

小白一句话

“帮我审一下前端改动，只说真正会影响质量的地方。”

什么时候用

改了 .tsx、.ts、.js 文件，想在提交前看有没有组件、性能、业务逻辑问题。

让 Codex 聚焦

告诉它审当前 diff，按紧急问题和改进建议分开列，不要泛泛点评。

最适合小白的结果

你会知道哪些问题值得马上修，哪些只是风格建议。

安全扫描：Safe / High Risk，使用前先看来源
05

code-reviewer

来源：google-gemini/gemini-cli

已安装到 ~/.agents/skills

安装命令

npx -y skills add google-gemini/gemini-cli -g --skill code-reviewer -y --full-depth

小白一句话

“请 review 当前改动，重点看 bug、边界条件、安全和测试缺口。”

什么时候用

任何代码改完后都可以用，尤其是跨文件、跨模块、涉及接口或数据状态的改动。

让 Codex 像 reviewer

要求它先给发现的问题，再给总结；每个问题要带文件和行号。

最适合小白的结果

你不用自己猜“能不能合”，Codex 会按正确性、可维护性、测试来过一遍。

安全扫描：Med Risk / Med Risk

webapp-testing

来源：anthropics/skills

已安装到 ~/.agents/skills

安装命令

npx -y skills add anthropics/skills -g --skill webapp-testing -y --full-depth

小白一句话

“启动页面后，用浏览器点一遍关键流程，截图并告诉我哪里坏了。”

什么时候用

你做了网页、表单、登录、按钮、列表、弹窗，想确认它真的能跑。

让 Codex 不只看代码

要求它开本地服务、等页面加载、用 Playwright 交互、看控制台报错。

最适合小白的结果

减少“代码看着对，页面一打开就崩”的情况。

安全扫描：Safe / Low Risk

小白一句话
